Editar perfil — hub + tela por campo

Em vez de uma parede de campos, Editar perfil vira um hub: cada linha mostra o valor atual e abre uma tela focada de um campo só (padrão Instagram). Uma responsabilidade por tela, contexto limpo, validação local e Salvar no header de cada uma. O hub agrupa por Perfil público · Preferências · Conta · Privacidade.
